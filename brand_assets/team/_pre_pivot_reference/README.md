## Warum diese Bilder hier liegen

Diese 3 Bilder wurden vor dem Brand-Pivot (07.05.2026 von "Norex AI / Prozessautomation" zu "Digitalisierungspartner für Handwerk + Reinigung") erstellt. Sie tragen entweder:
- Das alte Pre-Pivot-Logo "NOREX Prozessautomation" eingebrannt
- Die alte Pre-Pivot-Tagline ("Ihr zertifizierter Prozess- & Digitalisierungspartner")
